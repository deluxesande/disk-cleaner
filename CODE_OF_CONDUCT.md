# Code of Conduct

## Our Pledge

Disk Cleaner is maintained to be a useful, safe, and respectful open-source project. Contributors, users, maintainers, and participants are expected to help keep the project welcoming and professional.

We pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity or expression, level of experience, nationality, personal appearance, race, religion, sexual identity or orientation, or technology preferences.

## Expected Behavior

Participants are expected to:

- Communicate respectfully and constructively.
- Assume good intent, but also accept correction when behavior causes harm.
- Keep discussions focused on the project and the technical work.
- Give clear feedback without personal attacks.
- Respect different experience levels.
- Follow maintainer decisions once a decision has been made.
- Report security issues privately instead of opening public issues.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, intimidation, threats, or abusive comments.
- Personal insults or repeated hostile behavior.
- Discriminatory language or conduct.
- Sexualized language, imagery, or attention.
- Publishing private information without permission.
- Trolling, spam, or intentionally disruptive behavior.
- Pressuring maintainers to accept unsafe changes.
- Publicly disclosing security vulnerabilities before maintainers have had a reasonable chance to respond.

## Project Safety Standard

Disk Cleaner is a tool that may eventually scan and delete files. Because of that, discussions and contributions must treat user data safety seriously.

Contributors must not encourage unsafe deletion behavior, hidden destructive actions, bypassing confirmation flows, or changes that make it harder for users to understand what the tool will remove.

## Enforcement

The project owner, `deluxesande`, is responsible for interpreting and enforcing this code of conduct.

Maintainers may take any action they consider appropriate, including:

- Editing or removing comments
- Closing issues or pull requests
- Requesting behavior changes
- Blocking users from participation
- Reporting serious abuse to the relevant platform

## Reporting

If you experience or witness unacceptable behavior, report it to `deluxesande` through the appropriate private contact method for the repository or platform.

Reports should include:

- What happened
- Where it happened
- Who was involved
- Any relevant links, screenshots, or context

Maintainers will review reports as privately and fairly as possible.

## Scope

This code of conduct applies in all project spaces, including:

- Issues
- Pull requests
- Discussions
- Documentation
- Release channels
- Any public or private communication related to the project

## Attribution

This code of conduct is adapted from common open-source community standards and tailored for Disk Cleaner.
